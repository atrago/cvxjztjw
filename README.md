# 项目简介

## 核心功能
- 前后端代码
- SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
- 管理员模块：用户管理、图书馆管理、座位预约、取消预约、系统管理、个人中心

## 适用场景
- 图书馆座位预约管理
- 机构内部资源分配
- 优化图书馆使用效率

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
